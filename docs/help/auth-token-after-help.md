# Auth Token

Inspect or manage the saved JWT stored in the local Monk config.

Use this when you need to view the current session token, overwrite it
manually, or clear it from disk.

Common uses:

- `monk auth token get`
- `monk auth token set @token.txt`
- `monk auth token clear`

Use `--help` on `get`, `set`, or `clear` for the next level down.
